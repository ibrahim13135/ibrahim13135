## About Me

I'm an Agricultural Engineer (Ain Shams University) who taught himself backend development through ALX's Software Engineering program — not as a career pivot, but to fix problems I kept seeing in the field: no traceability, no data, no tooling built for how Egyptian farms actually work.

I build systems that sit at that intersection — IoT sensors, FastAPI backends, and data pipelines designed for irrigation, pest management, and agricultural export compliance.

- 🌱 Agricultural Engineering background — irrigation systems, pest management, crop monitoring
- 💻 Backend development — Python, FastAPI, Flask, Express.js
- 📡 IoT systems — sensor integration, automation, PLC logic
- 🗄️ Databases — PostgreSQL, MongoDB, Redis, SQLAlchemy
- 🤖 Applied ML — CNN-based plant disease detection, climate/evaporation modeling

## 🔭 Currently Building

**[NileTrace](https://github.com/ibrahim13135/niletrace)** — an agricultural export traceability platform for Egyptian SME exporters (dates, citrus, herbs). It tracks a harvest batch from the farm plot through processing, lab testing, and shipment, generating QR-traceable compliance documents along the way — built for exporters who currently do this on paper.

**Tech:** FastAPI · PostgreSQL + SQLAlchemy 2.0 · Alembic · JWT auth · ReportLab (PDF generation) · QR traceability · Streamlit (MVP frontend)
